Start Quiz Admin Panel

Admin Guidance

Manage Active Quiz

Activate a quiz for students to take. Only one quiz can be active at a time, and it will be shown to students once active.

Set Timers

Question Time: Alloted Time for each question.
Available Time: Total quiz duration. Students can take the quiz only within this period.

Results Viewing

Once the available time expires, students can view their results, including the questions they attended and the correct answers. Until then, they can only view their score.

Question Management

Admin can add, edit, or update questions for the active quiz. Questions can be added manually or uploaded via Excel/CSV files.

Shuffling Questions and Options

Questions and answer options can be shuffled to ensure fairness and prevent cheating.

Edit Quiz Settings

Admins can modify the start and end times for the quiz, as well as the time allocated for each question, at any point before the quiz begins.

Full-Screen Mode

Students must be in full-screen mode while attending the quiz for an immersive experience, ensuring they focus on the quiz without distractions.
